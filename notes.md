**ping #ip** --> Verificar se o PC está se conectando com o SERVER, ou seja, verifica se ele está na rede



Acessamos os sites pelo IP do SERVIDOR, porém o DNS faz o alias com o IP, facilitando a entrada. Ou seja, invés de toda vez que entrar no site utilizar o IP, usamos a URL.

Toda máquina responde ao IP 127.0.0.1 - Localhost. Usamos para testar o loopback localhost
Na ATV1, para fazer o telefone funcionar na rede, é preciso configurar o DHCP no servidor. No smartphone, além de configurar seu IP, temos que configurar o roteador.
